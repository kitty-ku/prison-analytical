This project was created as part of coursework for computer science at Gdańsk Univeristy of Technology.

I took particular enjoyment in creating the models included in the repository,
to an extent that I went out of my way to try and include as many UML elements as possible,
extending beyond the demands of the course.
I have worked with Spark System's documentation to make sure every element used is in the right place 
and feel comfortable navigating the ecosystem.

The HTML publish generated by EA is available as a [GitHub pages deployment](https://kitty-ku.github.io/prison-analytical/).

The model is composed of 8 diagrams which were created independently.
As an educational project and my first contact with the software, 
it involves solutions that I am not proud of
and would have done differently knowing better. 
Most importantly there are no common elements used between diagrams, 
so the models appear as separate.
